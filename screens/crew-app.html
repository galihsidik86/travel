CORE LOOP

Manifest → Tracker → Alert → Resolve

Crew memulai hari dari Bus Manifest. GPS tracker dijalankan otomatis ketika manifes "berangkat". Setiap anomali (SOS, jemaah keluar zona) muncul di Alert dengan prioritas terurut.

DATA SYNC

Real-time ke HQ & Owner Dashboard

Setiap check-in, SOS, atau update manifest langsung tercermin di Web Admin (Owner) dan Portal Agen yang relevan. Tidak ada lagi laporan H+1 via WhatsApp pribadi.

FALLBACK

Bekerja saat sinyal lemah

Cache lokal menyimpan manifest hari berjalan. SOS dikirim via SMS gateway jika 4G/5G tidak tersedia. Setiap aksi memiliki audit log yang sinkron saat sinyal kembali.
